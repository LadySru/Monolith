# 🪨 Monolith Social — Website Hosting Checklist

## What You Need (3 files, keep them together)
- [ ] `index.html` — the landing page
- [ ] `banner.jpg` — your server banner image
- [ ] `avatar.jpg` — your server profile picture

All 3 files must be in the **same folder** when uploaded.

---

## Option A — Netlify Drop (Easiest, Free, ~2 min)

- [ ] Go to **netlify.com/drop**
- [ ] Drag your folder (containing all 3 files) onto the page
- [ ] Your site goes live instantly with a link like `random-name.netlify.app`
- [ ] (Optional) Click "Claim your site" to rename it to something like `monolithsocial.netlify.app`
- [ ] Share the link on Reddit!

---

## Option B — GitHub Pages (Free, good for Reddit links)

- [ ] Create a free account at **github.com**
- [ ] Click **New Repository** → name it `monolith-social` → set to **Public**
- [ ] Upload all 3 files (`index.html`, `banner.jpg`, `avatar.jpg`)
- [ ] Go to **Settings → Pages → Source** → select `main` branch → Save
- [ ] Your site will be live at `yourusername.github.io/monolith-social`
- [ ] Wait ~1 minute for it to build, then share!

---

## Option C — Carrd (Easiest Custom Domain, ~$9/yr)

- [ ] Go to **carrd.co** and sign up
- [ ] Use "Embed" to paste your HTML code, or use their builder
- [ ] Great option if you want `monolithsocial.com` as a real domain

---

## Posting on Reddit

- [ ] Post in subreddits like **r/DiscordServers**, **r/MakeNewFriendsHere**, **r/animeclub**, or **r/gamingnews**
- [ ] In your Reddit post, paste your site link alongside the Discord invite
- [ ] The site's **og:image** (your banner) will show as a preview on Reddit automatically
- [ ] Title suggestion: *"Monolith Social — Cozy anime & gaming Discord, est. 2021 · Free to join"*

---

## Quick Checks Before Going Live

- [ ] Open the site in a browser — does the banner image load?
- [ ] Does the avatar appear in the nav and hero?
- [ ] Click the **Join Free** button — does it open the correct Discord link?
- [ ] Test on mobile (resize your browser or use phone)
- [ ] All 3 files uploaded to the same folder/repo ✓
